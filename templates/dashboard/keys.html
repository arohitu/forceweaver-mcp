Integration Guide
VS Code GitHub Copilot

Add to your VS Code settings.json:

{
  "github.copilot.chat.mcpServers": {
    "forceweaver": {
      "command": "python",
      "args": ["mcp_server/enhanced_server.py"],
      "env": {
        "FORCEWEAVER_API_KEY": "your_api_key",
        "SALESFORCE_ORG_ID": "production"
      }
    }
  }
}
Claude Desktop

Add to your Claude Desktop config:

{
  "mcpServers": {
    "forceweaver": {
      "command": "python",
      "args": ["mcp_server/enhanced_server.py"],
      "env": {
        "FORCEWEAVER_API_KEY": "your_api_key",
        "SALESFORCE_ORG_ID": "production"
      }
    }
  }
}
Environment Variables
Variable Description Example
FORCEWEAVER_API_KEY Your ForceWeaver API key fk_...
SALESFORCE_ORG_ID Your Salesforce org identifier production, sandbox
Important: Replace your_api_key with your actual API key and production with your Salesforce org identifier.
