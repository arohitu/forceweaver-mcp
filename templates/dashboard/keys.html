Integration Guide
VS Code GitHub Copilot

Add to .vscode/settings.json:

{
  "github.copilot.chat.mcpServers": {
    "forceweaver": {
      "command": "python",
      "args": ["mcp_server/enhanced_server.py"],
      "env": {
        "FORCEWEAVER_API_KEY": "your_api_key_here",
        "SALESFORCE_ORG_ID": "production"
      }
    }
  }
}
