Billing Information
Pay-Per-Use Model

ForceWeaver MCP uses a simple pay-per-use billing model. You only pay for the API calls you make.

Pricing
  • Basic Health Checks: $0.01 per call
  • Sharing Model Analysis: $0.01 per call
  • Bundle Analysis: $0.01 per call
  • Data Integrity Check: $0.01 per call
  • Performance Metrics: $0.05 per call (Premium)
  • Security Audit: $0.05 per call (Premium)

Usage Optimization Tips
Reduce Costs
  • Use specific check types instead of running all checks
  • Cache results when possible to avoid duplicate calls
  • Monitor failed calls and fix configuration issues
  • Use basic checks for regular monitoring, premium for deep analysis
Optimize Performance
  • Ensure Salesforce Connected Apps are configured correctly
  • Keep client secrets up-to-date
  • Use appropriate API versions for your Salesforce org
  • Monitor response times and report slow calls
